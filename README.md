# Frost Town Godot 3D Playtest

Temporary Godot 4.7.1 Web export for browser acceptance. This repository contains build output only, not the source project. It is not a production release.

Current exact-SHA candidate: `/v8/` from source `25e013525c44c4705cdd0f45f36d9295fb5a8b9e`.

V8 adds original procedural Blender GLB assets, resident allocation, capped offline income, hero team growth, the day-to-night defense vertical slice, a first-session interaction panel visible from spawn, safe-spawn progression saves, and a closed level-two loop whose objective resets after victory. It remains a temporary acceptance build, not a production release.
