# Frost Town Godot 3D Playtest

Temporary Godot 4.7.1 Web export for browser acceptance. This repository contains build output only, not the source project. It is not a production release.

Current exact-SHA candidate: `/v11/` from source `409721456921eeaa110af7190e6cdef8cdac567d`.

V11 rebuilds the original procedural 3D assets, fixes portrait composition, camera drag/zoom, touch ownership, HUD overlap, forward movement direction, and the complete upgrade-to-night-defense loop. It remains a temporary acceptance build, not a production release; physical phone and five-person human acceptance are still pending.

Legacy root, v2, v3, and v4 exports were removed after a security audit found non-secret governance metadata embedded in their PCK files.
